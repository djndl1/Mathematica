Set Theory

This node is not intended as a thourough treatment of set theory. It is only elements of naive set theory: set theory is regarded as a body of facts, of which the axioms are a brief and convenient summary.
General set theory is pretty trivial stuff really, but, if you want to be a mathematician, you need some, and here it is; read it, absorb it, and forget it.

a set R is a relation if each element of R is an ordered pair, denoted by x R y ⊂ X × Y

Remark. The empty set and the Cartesian product of any two sets are two examples. Another example can be the relation of equality that is the set of all those pairs ( x , y ) in X × X for which x = y .

A relation in R in X is reflexive if ∀ x ∈ X , x R x ; symmetric if x R y implies that y R x ; transitive if x R y and y R z imply that x R z . It is equivalent if it is reflexive, symmetric and transitive. The smallest equivalence relation in a set X is the relation of equality in X and the largest equivalence relation in X is X × X .

4 Function

Definition 1. function
A function from X to Y (codomain, target set) is a relation f s.t. dom f = X and s.t for each x in X there is a unique element y in Y with ( x , y ) ∈ f . The element y is called the value that the function f assumes (or takes on) at the argument x ; or we may say f sends or maps or transforms x into y .
Remark. The words map or mapping, transformation, correspondence, and operator are among some of the many that are sometimes used as synonyms for function. The set of all functions from X to Y is a subset of the power set P ( X × Y ) , and denoted by Y X . If dom f = Y , we say f maps X onto Y (surjective), denoted by f ( X ) = Y . The subset of all values to which from a subset A of X f maps is called the image, denoted by f ( A ) .
If X is a subset of a set Y , the function f defined by f ( x ) = x for each x in X is called the inclusion map (the embedding, or the injection) of X into Y . The inclusion map of X into X is called the the identity map on X .
Let X ⊂ Y , and f : Y ↤ Z , define g : X ↤ Z to be equal to f ( x ) for each x in X . The function g is called a restriction of f to X , and f is called an extension of g to X . This makes small functions out of larger ones.
The function f : X × Y ↤ X is called a projection onto the first coordinate, similarly for f : X × Y ↤ Y .
Suppose R is is an equivalence relation in X , and let f be the function from X onto X / R defined by f ( x ) = x / R . This function is sometimes called the canonical map from X to X / R .
A function that always maps distinct elements onto distinct elements is called one-to-one (injective)
Y X = { f | f : X → Y }
If A is a subset of a set X , the characteristic function of A is the function χ from X to 2 such that χ ( x ) = 1 or 0 according to x ∈ A or x ∈ X - A .

Proof. To prove this proposition, we need to prove
i. no natural number is a subset of any of its elements
By mathematical induction, let S be the set of all those natural numbers n that are not included in any of their elements. Obviously, 0 ∈ S . If n ∈ S , and n + = n ∪ { n } , suppose n + ⊂ x , then n ⊂ x , hence x ∉ n , otherwise n ∉ S , then n + cannot be a subset of any element of n , and if n + ⊂ n , then n ∈ n , but n ∉ n . Hence n + ∈ S .
ii. every element of a natural number is a subset of it. Or every natural number is transitive (includes everything that it contains, x ∈ y and y ∈ E , then x ∈ E )
Let S be the set of all transitive natural numbers. 0 ∈ S is vacuously true. Suppose n ∈ S . If x ∈ n + , then x = n or x ∈ n , in either case x ⊂ n + . Hence n + ∈ S .
Let prove the original proposition now. Since n ∈ n + , then n ∈ m + , and then n = m or n ∈ m . Similarly m = n or m ∈ n . If m ≠ n , then we must have n ∈ m and m ∈ n , from the second auxilliary proposition n ⊂ n , which contradicts the first auxilliary proposition.
The above five propositions are known as the peano axioms. From them it is possible to define intergers, rational numbers, real numbers and complex numbers and to derive their usual arithmetic and analytic properties.

A relation R in a set X is called antisymmetric if for every x and y in X , the simultaneous validity of x R y and y R x implies that x = y .

A partial order in a set X is a reflexive, antisymmetric and transitive relation in X .
Definition 3. A partial order in X may be defined as a relation ≦ in X s.t. for all x , y and z in X , we have
i. x ≦ x
ii. if x ≦ y and y ≦ x , then x = y
iii. if x ≦ y and y ≦ z , then x ≦ z .

The reason for the qualifying “partial” is that some questions abotu order may be left unanswered.
Definition 4. if for every x and y in X either x ≦ y or y ≦ x , then ≦ is called a total order. A totally ordered set is frequently called a chain.

e.g. inclusion in a power set is a partial order; “less than or equal to” in the set of natural number is a total order; the relation of extension for functions is a partial order.
Definition 5. a partially ordered set is an ordered pair ( X , ≦ ) , where X is a set and ≦ is a partial order in X .

This kind of definition is very common in mathematics; a mathematics structure is almost always a set together with soem specified other sets, functions, and relations. The accepted way of making such definitions precise is by reference to ordered pairs, triples or whatever is appropriate.
If x ≦ y and x ≠ y , x is a predecessor of y . Similarly we can define successor.
The connection between ≦ and < can be generalized to arbitrary relations. Given any relation R in a set X , we can define a relation S in X by writing x S y in case x R y but x ≠ y . S is a strict relation while R is the weak relation corresponding to S .
If X is a partially ordered set, then it could happen that X has an element a s.t. a ≦ x for every x in X . The antisymmetry of an order implies that if X has a least element, then it has only one. Similar for the greatest element.
